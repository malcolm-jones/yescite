## About The Project

This is an interactive [website](https://malcolmjones.pythonanywhere.com/) with tools for managing/exploring `.bib` files. 

**Tools**:
- YesCite: Tidy a `.bib` file by removing any unused entries according to your `.bbl` file.
- Convert `.bib` to `.csv`.
