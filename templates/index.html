YesCite

This tool tidies your .bib file by removing any unused entries in your project.

Requirement: You need to be using BibTeX. This generates a .bbl that says which .bib entries are being used. The tidied .bib file contains the entries that appear in the .bbl.

Find your .bbl on Overleaf: In Overleaf, to find your .bbl: click "Logs and output files", scroll down, click "Other logs and files", and click to download "output.bbl". If there is no .bbl file, try recompiling. If there is still no .bbl file, check if you are using BibTeX.
